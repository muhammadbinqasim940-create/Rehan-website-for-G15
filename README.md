# Rehan Website for G15

This project is a simple website designed for IMCB School.

## What it does
The website showcases important information about the IMCB School, including its mission, staff, and contact details. It provides an easy-to-navigate interface for students, parents, and visitors.

## Pages Included
- **Home**: Welcome page and main navigation (index.html)
- **About**: Details about the school (about.html)
- **Contact**: Contact information and ways to reach the school (contact.html)
- **Faculty**: List of school staff and their details (faculty.html)

## Styling
The design and layout of all pages is managed through the `style.css` file, ensuring a consistent and attractive look throughout the website.
